- 自定义注解
- 注解解析

**文件处理（XML解析的常用方法，JSON的数据类型）**

**设计模式（工厂模式，单例模式，代理模式，观察者模式，MVC模式）**

#### 学习方法

在这个阶段的重点就不是学习新的技术了，而是对已经学的东西进行巩固，针对面试要求进行学习。这个阶段是整个后端学习阶段中最为主要的阶段。
